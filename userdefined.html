Java User-Defined Exceptions

Java provides us the facility to create our own exceptions which are basically derived classes of Exception. Creating our own Exception is known as a custom exception or user-defined exception. Basically, Java custom exceptions are used to customize the exception according to user needs.

The User-Defined Exception is created using the throw keyword

Why do we need User-Defined Exceptions ?

It is because manier times, the user has certain logical constraints or cases which need to be handled given those cases are encountered.

This could be best understood with an example.

Given below is a code exhibiting User-defined exception, for a case where if value of a variable a is encountered with a value less than 0, an User-defined exception would be invoked.
